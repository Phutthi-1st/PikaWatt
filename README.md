## 📌 เกี่ยวกับโปรเจกต์ (About The Project)
**PikaWatt** เป็น Mobile Application ที่ถูกพัฒนาขึ้นเพื่อช่วยให้ผู้ใช้งานสามารถคำนวณอัตราการใช้พลังงานไฟฟ้าของเครื่องใช้ไฟฟ้าแต่ละชนิดได้อย่างแม่นยำ โดยจุดเด่นของแอปพลิเคชันคือความสามารถในการนำอุณหภูมิแวดล้อมจริงมาคำนวณร่วมด้วย (Heat Multiplier) รวมถึงมีระบบวิเคราะห์จุดคุ้มทุน (Break-even Analysis) เพื่อช่วยในการตัดสินใจเลือกซื้อเครื่องใช้ไฟฟ้าที่คุ้มค่าในระยะยาว

## ✨ ฟีเจอร์หลัก (Key Features)
* 🌡️ **Heat Multiplier Algorithm:** ระบบประมวลผลค่าไฟของเครื่องทำความเย็นที่แม่นยำยิ่งขึ้น โดยอ้างอิงจากอุณหภูมิแวดล้อมจริง
* ⚖️ **Break-even Simulator:** ฟังก์ชันเปรียบเทียบเครื่องใช้ไฟฟ้า 2 รุ่น พร้อมแสดงกราฟจุดตัดคุ้มทุน (Break-even Point) ระยะเวลา 5 ปีแบบเรียลไทม์
* 🏢 **Custom Electricity Rate:** อิสระในการปรับตั้งค่าไฟต่อหน่วย (Unit Rate) ตอบโจทย์กลุ่มผู้ใช้งานที่พักอาศัยในหอพักหรืออพาร์ตเมนต์
* 📊 **Cost Visualization:** สรุปยอดค่าใช้จ่ายที่เข้าใจง่ายในรูปแบบตัวเงิน (บาท) ทั้งแบบรายวัน รายเดือน และรายปี
